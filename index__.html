Build your own  PLATFORM 

SPOO is a framework that lets you build custom platforms easily and everywhere, using abstract objects. It comes with built-in features, like authorization, messaging, user handling.



Get Started   Github
1. Install

npm i spoojs
2. Define your server (Using OBJY extention)

const SPOO = require('spoojs');
const OBJY = require('objy');

OBJY.define({
    name: "object", 
    pluralName: "objects",
    storage: new MongoMapper('localhost').connect()
})

SPOO.REST({
    OBJY,
    port: 80
}).run()
3. Access from a client
                                    <script src="https://cdn.jsdelivr.net/npm/@spootechnologies/spooclient@0.0.13/index.js">
<script>
// Authenticate
spoo.io().auth('username', 'password', (data, err) => {})

// Add an object
spoo.io()
    .object({name "hello world"})
    .add((obj, err) => {})
</script>
